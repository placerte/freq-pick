## Summary

- Add a helper to build overlay context with validation against the spectrum grid.
- Validate CLI overlay inputs and warn when an empty context is supplied.
- Improve overlay legend coverage and band styling in the Matplotlib UI.

## Changes

- Add `OverlayContext.from_arrays` to validate overlays against `f_hz`.
- Warn when `OverlayContext` is provided without any arrays.
- Improve overlay length mismatch error wording.
- Load `.npz` overlays through `OverlayContext.from_arrays` in the CLI.
- Include the spectrum line in the overlay legend.
- Adjust overlay band colors and transparency.
- Document overlay context usage in the README.
- Add tests for warning behavior and minimal overlay rendering.
